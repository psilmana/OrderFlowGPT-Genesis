# Release Notes — 0.1.3

## Milestone 8: Time Axis Detector

This release implements the second real object detector: `TimeAxisDetector`.

## Highlights

- Detects only the horizontal time-axis region immediately below the detected chart.
- Emits a `DetectedObject` with `ObjectType.TIME_AXIS`, frame id, confidence, bounds, source, and metadata.
- Adds immutable `TimeAxisDetectorConfig` thresholds for height ratios, edge density, confidence, projection scoring, brightness transitions, chart-overlap rejection, horizontal alignment tolerance, and debug overlays.
- Integrates with `SequentialObjectDetectionPipeline` so `PriceAxisDetector` runs first and `TimeAxisDetector` runs second when both are registered; both successful detections automatically contribute to `DetectionGraph`.
- Extends debug overlay support to draw the time-axis rectangle without changing chart or price-axis overlays.

## Intentional Exclusions

Milestone 8 does not implement OCR. The detector identifies the axis region only, not timestamp labels. Timestamp OCR is intentionally deferred because text recognition needs separate contracts, locale/timezone handling, confidence validation, and semantic parsing after deterministic geometry is stable.

Milestone 9 begins Footprint Grid Detection.
