# Release Notes — 0.1.13 Milestone 19

Milestone 19 adds deterministic Absorption Detection.

## Added

- Immutable absorption enums, configuration, detection, result, and statistics models.
- `FootprintAbsorptionDetector` for deterministic absorption observations derived from the immutable `FootprintMatrix` and existing `FootprintImbalanceResult`.
- Detection graph integration and lookup/statistics helpers for buy and sell absorption observations.
- Pipeline integration after footprint matrix, single-cell imbalance, and stacked imbalance construction.
- Regression tests for successful detections, thresholds, validation, immutability, graph integration, and pipeline behavior.

## Scope Guardrails

Milestone 19 remains deterministic and local. It adds no AI, ML, OCR changes, OpenCV additions, networking, async processing, threading, vendor-specific logic, trading signals, entries, exits, or recommendations.
