# Release Notes 0.1.5 - Milestone 10 Footprint Cell Grid Detection

- Added `FootprintCellDetector` for deterministic segmentation of a detected footprint grid into individual footprint-cell rectangles.
- Added immutable `FootprintCellDetectorConfig` with minimum cell size, spacing variation, alignment deviation, confidence, and debug-overlay settings.
- Extended detection results and the sequential object detection pipeline so all detected footprint cells are included in the validated `DetectionGraph` after chart, price axis, time axis, and footprint grid detections.
- Extended debug overlays to draw the footprint grid and every detected cell rectangle.
- This milestone detects only footprint-cell geometry. It does not read numbers, perform OCR, classify bid/ask, recognize volume, calculate delta, use AI/ML, or use OpenCV.
- Milestone 11 introduces the Cell Coordinate System.
