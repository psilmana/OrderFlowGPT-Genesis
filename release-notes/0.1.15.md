# Release notes 0.1.15

Milestone 21 introduces deterministic Volume Cluster Analysis. Genesis now classifies each individual `FootprintMatrix` cell by total cell volume using immutable percentile thresholds.

## Added

- Immutable `VolumeClusterType`, `VolumeClusterConfiguration`, `VolumeCluster`, `VolumeClusterResult`, and `VolumeClusterStatistics` models.
- `VolumeClusterAnalyzer`, which reads only immutable semantic values from `FootprintMatrix` and classifies cells as `HIGH_VOLUME`, `LOW_VOLUME`, or `NORMAL_VOLUME`.
- `DetectionGraph.volume_clusters` plus high-volume, low-volume, normal-volume, lookup, and statistics helpers.
- Pipeline integration immediately after `FootprintDeltaAnalyzer`.

## Scope

Milestone 21 classifies individual cell volume only. It does not implement Point of Control, HVN/LVN zones, market profile, auction logic, market bias, trading signals, entries, exits, or recommendations. Milestone 22 introduces Point of Control (POC).
