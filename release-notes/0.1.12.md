# Release Notes 0.1.12 - Milestone 18

Milestone 18 introduces deterministic stacked footprint imbalance detection.

## Added

- Added immutable `StackedImbalanceType`, `StackedImbalanceConfiguration`, `StackedImbalance`, `StackedImbalanceResult`, and `StackedImbalanceStatistics` models.
- Added `StackedImbalanceDetector`, which reads only the immutable Milestone 17 `FootprintImbalanceResult` and immutable `FootprintMatrix`.
- Added graph and pipeline integration so `DetectionGraph.stacked_imbalances` is populated after `FootprintImbalanceDetector`.
- Added helpers for stack lookup, cell lookup, ask/bid stack filtering, and statistics.

## Scope

Milestone 18 detects stacked bid/ask imbalances only. It adds no absorption detection, unfinished auction detection, POC detection, trading logic, trading signals, trade recommendations, AI, ML, OCR changes, OpenCV additions, networking, threading, async execution, or image processing.

Milestone 19 introduces Absorption Detection.
