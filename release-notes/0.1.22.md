# Release Notes 0.1.22 - Bundle 9 Dataset Builder & Annotation Infrastructure

Bundle 9 creates the canonical immutable training dataset layer for Genesis. Each analyzed frame becomes one complete training sample containing frame identity, timestamp metadata, deterministic Bundle 1-8 analysis references and summaries, Fabio annotation placeholders, dataset versioning, validation, statistics, serialization, and deterministic export capability.

Supported exports are JSONL, SQLite, and Parquet-compatible payload files. The pipeline is `DetectionGraph -> FeatureVectorBuilder -> TrainingSampleBuilder -> DatasetBuilder`.

Bundle 9 is AI-ready infrastructure only. It performs no machine learning, AI reasoning, neural-network training, LLM calls, prediction, probabilities, strategy generation, vendor-specific logic, OCR modifications, or OpenCV changes. Bundle 10 introduces the Learning Engine.
