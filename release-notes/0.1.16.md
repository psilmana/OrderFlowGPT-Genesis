# 0.1.16

## Genesis Market Profile Core (Bundle 1)

- Added immutable Session Point of Control (POC) analysis with deterministic tie-breaking.
- Added deterministic High Volume Node (HVN) and Low Volume Node (LVN) row analysis using configurable percentile thresholds.
- Added deterministic Value Area (VAH / VAL) analysis with a default 70% coverage target and configurable strict mode and minimum rows.
- Extended `DetectionGraph` lookup helpers, statistics, and sequential pipeline integration for the Market Profile Core.

Not implemented in this release: developing POC, developing value area, composite profile, Auction Theory, market prediction, or trading signals. Bundle 2 begins Auction Market Theory.
