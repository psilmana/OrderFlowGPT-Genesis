# Release Notes — 0.1.2

## Milestone 7: Price Axis Detector

This release implements the first real object detector: `PriceAxisDetector`.

## Highlights

- Detects only the vertical price-axis region immediately to the right of the detected chart.
- Emits a `DetectedObject` with `ObjectType.PRICE_AXIS`, frame id, confidence, bounds, source, and metadata.
- Adds immutable `PriceAxisDetectorConfig` thresholds for width ratios, edge density, confidence, projection scoring, brightness transitions, overlap rejection, and debug overlays.
- Integrates with `SequentialObjectDetectionPipeline` so the price-axis detector runs first when registered and automatically contributes to `DetectionGraph`.
- Extends debug overlay support to draw the price-axis rectangle without changing chart overlays.

## Intentional Exclusions

Milestone 7 does not implement OCR. The detector identifies the axis region only, not the price numbers. OCR is intentionally deferred because text recognition requires separate contracts, validation, and error handling after deterministic region geometry is stable.

Milestone 8 will detect the Time Axis next.
