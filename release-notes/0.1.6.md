# Release Notes 0.1.6 - Milestone 11

Milestone 11 introduces the complete logical coordinate system for footprint cells. Genesis now assigns every detected footprint cell a deterministic row index, column index, cell id, and grid id after footprint-cell geometry is available.

The release adds immutable `CellCoordinate`, `GridCoordinateSystem`, `CellReference`, and `CoordinateMapper` objects; lookup helpers for cells, rows, columns, ids, and neighbors; and optional coordinate-system exposure on `DetectionGraph` for downstream milestones.

No OCR is performed. No numbers are read or interpreted. No bid/ask, volume, or delta classification is implemented in this milestone. Milestone 12 introduces Cell Classification.
