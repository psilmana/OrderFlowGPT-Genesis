# Release Notes - 0.1.7

## Milestone 12 - Footprint Cell Classification

Milestone 12 teaches Genesis the logical structure of every detected footprint cell. After footprint cells are mapped by the coordinate system, the new `CellLayoutAnalyzer` creates deterministic, configurable semantic regions for each cell and returns immutable `CellClassification` results.

### Added

- `CellSemanticRole` enum values for unknown, bid, ask, delta, center, background, and empty logical regions.
- Immutable `CellRegion`, `CellLayout`, `CellLayoutAnalyzer`, and `CellClassification` models.
- Validation for duplicate roles, overlapping semantic regions, invalid confidence, invalid parent references, out-of-cell geometry, negative/zero geometry, and missing required regions.
- Metadata for cell id, row, column, semantic regions, and classification confidence.
- Logical helper methods for bid, ask, center, role-based, and all-region lookups.
- `DetectionGraph.cell_classifications` and `DetectionGraph.CellClassifications` exposure after footprint cell mapping.

### Explicitly out of scope

Milestone 12 performs no OCR, no text recognition, no numeric recognition, no bid/ask value recognition, no delta calculation, no volume calculation, no AI, no ML, no OpenCV processing, no networking, no threading, and no async execution.

Milestone 13 introduces OCR Foundation.
