# Release 0.1.25 — Bundle 13.5 Genesis Runner & CLI

Bundle 13.5 introduces the executable Genesis application. The permanent deterministic entry point is now available with:

```bash
python -m orderflowgpt_genesis --video assets/fabio/videos/Lesson01.mp4 --transcript assets/fabio/transcripts/Lesson01.txt
python -m orderflowgpt_genesis --folder assets/fabio/videos --output assets/fabio/output --overwrite
```

The runner creates lesson outputs containing frames, detections, dataset, knowledge, memory, `report.json`, `summary.json`, and `processing.log` without overwriting unless requested. Bundle 14 introduces Replay & Coaching.
