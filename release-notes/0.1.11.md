# Release Notes - 0.1.11 Milestone 17

Milestone 17 introduces deterministic single-cell Footprint Imbalance Detection. Genesis now evaluates the immutable `FootprintMatrix` produced by Milestone 16 and emits individual bid/ask imbalances with stable ordering, lookup helpers, and aggregate statistics.

## Added

- Immutable imbalance models: `ImbalanceType`, `ImbalanceSide`, `ImbalanceConfiguration`, `FootprintImbalance`, `FootprintImbalanceResult`, and `ImbalanceStatistics`.
- `FootprintImbalanceDetector` for deterministic ask-vs-bid-below and bid-vs-ask-above comparisons using parsed numeric values already present in the matrix.
- DetectionGraph integration through `footprint_imbalances`, imbalance lookup helpers, and statistics.
- Pipeline integration so `SequentialObjectDetectionPipeline` runs imbalance detection after `FootprintMatrixBuilder`.

## Scope

Milestone 17 detects only individual bid/ask imbalances. It adds no stacked imbalance detection, absorption detection, unfinished auction detection, POC detection, trading signals, strategy logic, trade recommendations, AI, ML, OCR changes, OpenCV additions, networking, threading, or async execution.

Milestone 18 introduces stacked imbalance detection.
