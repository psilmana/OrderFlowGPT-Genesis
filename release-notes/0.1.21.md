# Release Notes - 0.1.21

Bundle 6 implements the Genesis Trend Engine.

- Added deterministic Trend State Detection, Pullback Detection, Break of Structure, and Change of Character models, configurations, statistics, results, analyzers, and detectors.
- Extended `DetectionGraph`, public exports, and `SequentialObjectDetectionPipeline` with Trend Engine results, lookup helpers, and statistics helpers while preserving previous APIs.
- The Trend Engine uses only immutable objects produced by previous bundles, especially Market Structure and the immutable footprint matrix.
- Genesis still does not perform AI reasoning, prediction, trading signals, probability estimation, machine learning, OCR modifications, OpenCV changes, networking, threading, async work, randomness, or vendor-specific logic.
- Bundle 7 introduces Session Intelligence.
