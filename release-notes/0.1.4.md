# Release Notes 0.1.4 - Milestone 9 Footprint Grid Detection

- Added `FootprintGridDetector` for deterministic detection of the rectangular footprint grid inside the chart.
- Added `ObjectType.FOOTPRINT_GRID` and immutable `FootprintGridDetectorConfig`.
- Extended sequential object detection ordering so price axis runs first, time axis second, and footprint grid third when registered.
- Extended debug overlays to draw footprint-grid rectangles.
- This milestone detects only the footprint grid; individual footprint cells, bid/ask numbers, volume, delta, imbalance, OCR, AI, ML, and OpenCV remain out of scope.
- Milestone 10 will detect individual footprint cells.
