# OrderFlowGPT Genesis 0.1.0

Release date: 2026-07-18

## Milestone 1 complete

This release freezes the initial architecture for OrderFlowGPT Genesis. It establishes a deterministic Python package with validated immutable domain models, a stateless order-flow analyzer, complete documentation, and automated tests.

## Included capabilities

- Validated order-book levels, trades, and market snapshots.
- Deterministic spread and mid-price calculations.
- Deterministic book imbalance and trade imbalance calculations.
- Directional bias classification with confidence scoring.
- Public API exports for supported Milestone 1 types.

## Operational notes

The release is intentionally local and deterministic. It does not include live market data ingestion, broker connectivity, persistence, or language-model integrations.
