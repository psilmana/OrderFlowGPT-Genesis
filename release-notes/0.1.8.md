# Release Notes - 0.1.8

## Milestone 13 - OCR Foundation

Milestone 13 introduces the provider-neutral OCR Foundation. Genesis can now create OCR requests for predefined semantic cell regions, execute an OCR engine contract, and attach raw OCR results to the detection graph after cell classification.

### Added

- Immutable OCR models: `OCRRequest`, `OCRResult`, `OCRWord`, `OCRLine`, `OCRPage`, `OCRRegion`, `OCRConfiguration`, and `OCRMetadata`.
- Abstract OCR contracts: `OCREngine`, `OCRPipeline`, and `OCRProvider`.
- `DummyOCREngine` for deterministic architecture testing without real OCR.
- `SequentialOCRPipeline` that converts classified cell regions into OCR requests and returns ordered raw OCR results.
- `DetectionGraph.ocr_results`, `DetectionGraph.region_text(role)`, and `DetectionGraph.lookup(cell_id)` raw lookup helpers.
- OCR helpers for `words()`, `lines()`, `text()`, and `average_confidence()`.

### Scope boundaries

No OCR provider is implemented. Milestone 13 performs no numeric interpretation, no bid/ask recognition, no volume parsing, no delta calculation, no validation of extracted values, no AI, no ML, no networking, and no cloud OCR. Milestone 14 introduces OCR Post Processing.
